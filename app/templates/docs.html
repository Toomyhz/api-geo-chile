Documentación

Esta API devuelve información en formato JSON sobre regiones, provincias y comunas de Chile.

{% include "components/seccion_recursos.html" %}

Endpoints

  • GET /api/regiones → Listado de todas las regiones
  • GET /api/regiones/{codigo} → Detalle de una región
  • GET /api/regiones/{codigo}/provincias → Provincias de esa región
  • GET /api/provincias/{codigo}/comunas → Comunas de esa provincia
  • GET /api/comunas/{codigo} → Detalle de una comuna

Ejemplo

$ curl https://api-geo-chile.cl/api/regiones/13
{
  "codigo": "13",
  "tipo": "region",
  "nombre": "Metropolitana de Santiago",
  "lat": null,
  "lng": null,
  "url": null,
  "codigo_padre": null
}
