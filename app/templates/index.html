API División Político Administrativa de Chile

Descripción

Esta API proporciona información sobre la división político-administrativa de Chile, incluyendo regiones, provincias y comunas.

Recursos

Cada consulta a la API devuelve un JSON con la información solicitada. Cada recurso tiene la siguiente estructura:

Campo Descripción
Código Identificador único del sector.
Tipo Tipo de sector (región, provincia o comuna).
nombre Nombre del sector (región, provincia o comuna).
lat Latitud del sector.
long Longitud del sector.
url Url del sitio web del sector. (Municipalidad, etc.)
codigo_padre Código del sector padre (por ejemplo, el código de la región para una provincia).
  • /regiones
  • /provincias
  • /comunas
